# Indicators-Blocklists

  This is a list of IP address and domain indicators. It includes indicators related
  to command-and-control (C2) frameworks as well as those observed in scanning,
  brute-force activities and exploitation attempts. All indicators were observed within the last 7 days. They
  can be used in blocklists or alerting rules.

  These indicators are sourced and curated through **Torch**, a threat intelligence
  platform that collects, enriches, and tracks malicious infrastructure. Learn more
  at https://cskittens.com.

  ## About Torch

  [Torch](https://cskittens.com) is a threat intelligence platform focused on
  surfacing active malicious infrastructure. The indicators in this repository are a continuously refreshed
  snapshot drawn from Torch's dataset.
